This is a web-application developed by Vipul Cariappa N T and Sharmada Iyer.

This web site will help you manage your daily expenses. Login or Register In upper right corner to use the facilities provided by this web site.

A registered user can form groups and share its name and password with others to join the group. Members in a group can add, edit and delete bill's created. Creator of a group has the ablity to remove a existing member from the group or/and delete the group.

This site has some security measures implemented, but does not granty security of your data.

This site should only be used for personal use. Using this web-application for commercial purposes is not recommended by the developer.

How to use this web-application?
First, register as a new user. You can do this by clicking the register button in the upper right corner. Next, go to My-Groups from navbar and add-group from the drop down to create a new group. Enter the desired name and password for your group. You can ask others like your family members or your organisation members to join the same group by entering the group name and password in join-group page. You can access your newly created group from the same drop down. You can access all your entries here. You may find none in beginning. You can click the add bill button to create a new entry. In the create new bill page you need to add a new category before you can enter a new bill.
Now you can start you work.
